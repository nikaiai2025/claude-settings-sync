# CSVマージスキル

指定ディレクトリ内の複数CSVファイルを1つにマージします。

## 実行手順

1. ユーザーから対象ディレクトリのパスを受け取る（$ARGUMENTS から取得、未指定なら確認する）
2. 以下のコマンドでマージスクリプトを実行する:

```
python "C:\Users\user\.claude\skills\merge-csv\scripts\merge_csv.py" <ディレクトリパス> [出力ファイルパス]
```

3. 実行結果をユーザーに報告する

## 仕様

- 対象: 指定ディレクトリ直下の全 `.csv` ファイル（ファイル名順）
- ヘッダ: 1つ目のファイルのヘッダ行のみ保持。2つ目以降のヘッダは削除
- **「ファイル名」列をヘッダ先頭に追加**し、各データ行に元ファイル名（拡張子なし、カンマ等はトリミング）を付与
- **ヘッダ不一致時はエラーを出して処理を中断する**（どのファイルのどの項目が異なるか表示）
- 文字コード自動判定: UTF-8 BOM / UTF-8 / cp932(Shift_JIS) に対応
- 出力は **UTF-8 BOM付き** でExcelでそのまま文字化けなく開ける
- 出力ファイル名を省略した場合は `<実行日時YYYYMMDD_HHMMSS>_<ディレクトリ名>.csv` として同ディレクトリに保存
- 出力ファイル自体はマージ対象から自動除外（繰り返し実行可能）
- CSVフォーマットは問わない（任意の項目構成に対応）
